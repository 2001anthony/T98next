■はじめまして

　この度は、PC9821 Emulator T98-Next をダウンロードして頂き、誠にありがと
　うございます。
　次世代PC9821エミュレータのベース環境を実現するため開発しました。
　本エミュレータは、ハードウエアエミュレータを自由に組み合わせ、無限の拡
　張性を提供することを目標としています。
　まだまだ、不具合、速度面など、問題を多分に含んでおりますが、最終的に完
　全な実機同様の作動が出来るよう、頑張っていきたいと思います。 

　　　　　　　　　　　　　　　　　　　　　　　　　　　　(YUKI 2000/06/29)

■動作環境

　Microsoft Windows 95/98/Me/NT4.0/2000 (x86のみ)
　Direct X 3.0 以上
　65536色(16bit)以上表示可能なVideoCard(256色モードでは起動しません)
　PentiumII  300MHz 以上
　　（推奨）PentiumIII 500MHz 以上
　　　　　　V86モニタ(EMM386,VEM486等)使用時 PentiumIII 800MHz 以上
　Memory64MB以上（推奨）

■本版について

　本体、デバイスの仕様変更、バグ修正を行っています。
　主な修正個所は以下のとおりです
　・Next.exe
　　　MMU修正
　　　CPU修正
　　　16Mシステム空間修正
　　　起動オプション（INI指定）追加
　　　INI仕様修正
　・GDC
　　　拡大処理修正
　　　GRCG修正
　　　CG-Window修正
　　　BIOS修正
　　　1280x960画面モードを追加
　・DISK
　　　DMAC修正
　　　I/O選択機能修正
　　　FDC修正
　　　D88のラベルを表示できるようにした

■T98-NEXT旧版からのアップグレード

　main.ini および SAVEフォルダを削除してください。
　これらは、旧版とは互換性がありません。
